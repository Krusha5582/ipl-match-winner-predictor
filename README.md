# IPL Match Winner Predictor

## 🚀 Overview

This project predicts IPL match winners using Machine Learning.

## 🛠 Tech Stack

* Backend: Python, FastAPI
* Frontend: React (Vite)
* Database: SQLite
* ML: Scikit-learn, Pandas

## 📁 Project Structure

* backend/ → API + ML models
* frontend/ → UI
* data/ → datasets
* notebooks/ → experiments
